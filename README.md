#The Vendor API - Inventory Hub 
This is a SaaS solution that enables trading partners to efficiently manage their connected inventory.
I developed the site's authentication and authorization module, along with configurations for email, SMS, and Stripe payment gateway integration. I also developed the onboarding module, where trading partners sign up for the system, upload documents for verification, and utilize a review and approval mechanism through SignalR. This mechanism allows users with the 'System Admin' privilege to review submitted documents and make a verdict.
